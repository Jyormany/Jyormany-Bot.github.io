  • Jyormany bot について

    YouTubeチャンネル「おんりー / ドズル社」の一部の生配信を、Twitterでのツイートで通知するBotアカウントです。

    メンバーシップ限定配信の通知がYouTubeアプリから届かないことがあったため、2022年2月6日からドズル社ファンの Jyormany (@Jyormany) が運営しています。

    通知対象として設定中の再生リスト(⬇︎お知らせ中の再生リスト)に新しく追加された配信枠のうち、配信前または配信中のものが見つかると、ツイートでお知らせします。
    (設定されている再生リストに含まれない配信は通知することができません。ご了承ください。)

    ただし、深夜1時から朝7時前までに検知した配信予定(配信枠の追加)は、すぐに通知はせず、朝7時以降に通知します。配信開始の通知は、24時間通知します。

    通知対象の再生リストの変更など、Botに関するお知らせがある場合は、通常は次の配信通知ツイートの本文に含めてお知らせします。また、通知・お知らせ以外のツイートは致しません。

  • このBotの仕組みの概説

    1. 再生リストにある動画の情報をすべて取得
    2. ⬇︎
    3. いままで通知していないか確認
    4. ⬇︎
    5. 非公開設定でなく、配信前または配信中のものに絞り込み
    6. ⬇︎
    7. 文章を組んでツイート
    このプログラムを、以下のサービスを利用して約2分3分おきに実行しています。
    • Google Cloud Platform (YouTube API)
    • Twitter API
    • Heroku / Heroku Scheduler (← 他サービスへ引っ越し予定)
    • GitHub
    実行中のプログラム(Python)をGitHubで公開しています。
